## Syntax

```shell
set prompt \001\033[COLORm\002TEXT \001\033[0m\002
set prompt \001\033[FORMATm\002TEXT \001\033[0m\002
set prompt \001\033[FORMAT;COLORm\002TEXT \001\033[0m\002
```
