Topic/Consumer
==============

- Relationship between consumers and partitions is one-to-many.
- Messages are consumed in FIFO manner.

.. code-block:: text

                    Topic 1                                 Consumer Group 1
    +-------------------------------------+               +------------------+
    |                                     |               |                  |
    |              +---+---+---+---+---+  |               |  +------------+  |
    |  Partition 0 |   |   |   |   | X +------+------------->+ Consumer 1 |  |
    |              +---+---+---+---+---+  |   |           |  +------------+  |
    |                                     |   |           |                  |
    |              +---+---+---+          |   |           |  +------------+  |
    |  Partition 1 |   |   | X +------------------+--------->+ Consumer 2 |  |
    |              +---+---+---+          |   |   |       |  +------------+  |
    |                                     |   |   |       |                  |
    |              +---+---+---+---+      |   |   |       |  +------------+  |
    |  Partition 2 |   |   |   | X +------------------+----->+ Consumer 3 |  |
    |              +---+---+---+---+      |   |   |   |   |  +------------+  |
    |                                     |   |   |   |   |                  |
    +-------------------------------------+   |   |   |   |  +------------+  |
                                              |   |   |   |  | Consumer 4 |  |
                                              |   |   |   |  +------------+  |
                                              |   |   |   |                  |
                                              |   |   |   +------------------+
                                              |   |   |
                                              |   |   |
                                              |   |   |
                                              |   |   |     Consumer Group 2
                                              |   |   |   +------------------+
                                              |   |   |   |                  |
                                              |   |   |   |  +------------+  |
                                              |   +-->+----->+ Consumer 1 |  |
                                              |           |  +------------+  |
                                              |           |                  |
                                              |           |  +------------+  |
                                              +------------->+ Consumer 2 |  |
                                                          |  +------------+  |
                                                          |                  |
