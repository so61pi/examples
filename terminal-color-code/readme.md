## Code sequence

```txt
Set format or color           \033[NUMm
                              \033[NUM(;NUM)*m
Reset all format and color    \033[0m
```

## Format

```txt
Bold              1
Dim               2
Italic            3
Underlined        4
Blink             5
Reverse           7
Hidden            8
```
